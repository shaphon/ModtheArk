ModtheArk ver3.1.0
非mod开发者仅需要复制dll文件



1.允许通过修改gdata导入
①buff图标
②技能卡图
③角色被动图标，图鉴页面封面，立绘等
④随机事件的配图
2.修改gdata时请按照格式新增，注意不要删除旧有的词条（可以设置词条内容为空）
3.角色立绘和战斗外头像，以及一切以来GameObject的东西，需要针对各个角色mod自行导入并且调整所导入图片的大小、偏移量、中心点等
4.本mod内置了开局选角色界面，如果可选角色>20，则会把头像框以相似比1/2缩小的功能
5.本mod内置了外部图片和自建类导入功能，使用方法请看相关txt
6.本mod也可以导入游戏内部图片，相关使用方法请看txt
7.本mod可以导入Unity的AssetBundle，相关使用方法请看txt
8.在AssetBundle导入音乐的基础上，可以直接调用函数来播放一次音乐（音量与系统一致），相关使用方法请看txt
